# lab-cn-page

本仓库用于归档和发布多个独立 HTML 页面。页面内容涉及不同主题，彼此之间无直接关联。仓库主要目的是提供静态页面的集中存放与访问入口，不针对任何特定域名或网站。

## 目录说明

- `pages/` — 存放所有 HTML 页面文件，每个文件对应一个独立页面。
- `assets/` — 存放页面引用的静态资源（如 CSS、JavaScript、图片等）。
- `index.html` — 仓库根目录下的入口页面，可提供页面索引或导航。

## 页面归档说明

归档的 HTML 页面均为独立文件，不依赖外部数据库或后端服务。页面可能包含简单的交互功能，但整体以静态内容为主。每个页面均可在浏览器中直接打开运行。

归档页面会不定期更新或移除，不保证所有页面长期可用。仓库不承担页面内容的更新责任。

## 维护说明

- 仓库由维护者自主管理，不开放外部提交。
- 如发现页面无法正常访问或存在明显错误，可通过 Issues 反馈。
- 仓库不提供任何形式的保证或支持服务。

## 使用方式

1. 直接打开根目录下的 `index.html` 查看页面列表。
2. 或直接访问 `pages/` 目录下的具体页面文件。

## 许可

本仓库内容仅供个人参考与学习使用。具体页面可能引用第三方资源，其版权归原作者所有。
